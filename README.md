# azi

Show Azure information.

## Installation

If you have Cargo installed, you can use `cargo install azi`

## Usage

List all subscriptions and resource groups:

```
azi list
```

Show the costs of March 2019:

```
azi costs 201903
```

Show DNS entries and resource groups they point to:

```
azi domains
```
